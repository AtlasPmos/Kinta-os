# KINTA OS v1.1

Stable usable release for GitHub Pages.

This version keeps all files in the repository root so iPad upload is easier.

Includes:
- Daily PM dashboard
- Project Command Center
- Clickable 2D equipment map
- Clickable 3D mock kitchen
- Equipment records
- Startup checklist
- Issue tracker
- Build history timeline
- Warranty status
- Document/manual links
- Export/import backup
- Local iPad browser saving
