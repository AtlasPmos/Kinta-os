## Upload instructions

Upload the contents of this folder to the root of the GitHub repository.

Upload these files:

- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`
- `README.md`
- `CHANGELOG.md`
- `ROADMAP.md`
- `VERSION.json`

Do not upload the zip file itself.
Do not upload the folder as a folder.
Do not leave an old broken manifest in place.

If Pages fails again, delete `manifest.json` and `sw.js` from GitHub first, wait for deploy, then upload this full set again.
