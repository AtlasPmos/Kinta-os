# KINTA Genesis Mod 8 — iPad PWA 404 Fix

This build fixes the most common issue where the GitHub Pages site works in Safari but the installed iPad home-screen app opens a 404 page.

## Upload to GitHub
Upload these root files only:

- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png
- README.md
- CHANGELOG.md
- ROADMAP.md

Do not upload the zip. Do not upload the containing folder.

## iPad fix
1. Delete the old KINTA home-screen app from the iPad.
2. Open the working GitHub Pages root URL in Safari.
3. Tap Share > Add to Home Screen.
4. If it still shows 404, clear Safari website data for the GitHub Pages domain and repeat.
