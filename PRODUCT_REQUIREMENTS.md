# KINTA Enterprise v1.0 Product Requirements

## Goal
Create a commercial kitchen PM operating system that combines project management, equipment intelligence, MEP/startup readiness, document memory, AI recommendations, and reporting.

## v1.0 Must Work
- Projects
- Equipment DNA
- MEP verification
- Upload Vault metadata
- Ask KINTA with live AI when backend key is configured
- Daily Brief
- Action Center
- Reports/email drafting
- Audit log
- Cloud deployment path

## Not v1.0 Yet
- Full OCR
- True image recognition
- WebGL digital twin
- Billing
- Multi-tenant production security review

## Success Definition
Rick can open KINTA, select Wylie/Galaxy/Waller Creek, ask what is blocking work, generate an email/report, and update equipment/action data without starting over.
