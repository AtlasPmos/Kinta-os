# Roadmap

## Milestone 1: Product Foundation
Single repo, frontend/backend/database, seeded PM data, deployment guide.

## Milestone 2: Daily Work Usability
Editable actions, equipment records, MEP grid, brief/report generation, file metadata upload.

## Milestone 3: Document Intelligence
PDF/text extraction, embeddings, retrieval, citations, conflict detection.

## Milestone 4: Digital Kitchen
Canvas floor plan, equipment blocks, dimensions, rotation, Equipment DNA linking.

## Milestone 5: Commercial Platform
Auth, organizations, permissions, audit history, subscriptions, support tooling.
